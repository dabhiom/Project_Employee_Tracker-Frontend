# Project Manager Tracker

## 📌 Overview

A centralized system to manage Employees, Projects, Resource Allocation, Finance (PO), Leave Tracking, and Bot Warehouse.

## 🛠 Tech Stack

* Frontend: React (Vite)
* Backend: Node.js + Express
* Database: MongoDB

## 🚀 Features (MVP)

* Role-Based Authentication
* Employee Management
* Project Management
* Resource Allocation
* Leave Tracking
* PO Tracking
* Bot Warehouse
